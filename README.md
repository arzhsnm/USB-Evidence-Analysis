# USB-Evidence-Analysis
A cat has been kidnapped by an unknown group. A suspect was arrested carrying a USB drive. Your task is to analyze the evidence and determine the city where the cat is being held. Provide the city name in lowercase.

Из образа USB извлекли файлы через foremost.

Нашли изображение с котом.

Из метаданных (EXIF) извлекли GPS-координаты.

Координаты ведут к коммуне Эльфранцкирш (Helfrantzkirch) во Франции, в регионе Эльзас.

Именно это название и является ответом.
